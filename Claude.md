# Jordan Brand Container Automation

Project Overview
Rule-based Python automation that monitors open Jordan Brand orders in Microsoft Fabric Gold, calculates container volume daily, and sends shipment report emails automatically via Microsoft 365.
Jira Epic: INT2-50
Build Ticket: INT2-411
Owner: Michael Sahadi (msahadi@maxwoodfurniture.com)

Key Context

This is not an AI project — pure rule-based automation. No NLP, no ML.
Jordan Brand is a furniture retailer (Jordan's Furniture, MA). Maxwood ships via full container loads (~2,800 cf).
The automation replaces a manual weekly/bi-weekly process Heather Collins currently does by hand.
Beverly Chartier (bchartier@jordans.com) is the Jordan contact — she approves shipments via email reply.
All data comes from Microsoft Fabric Gold — no SalesPad API, no SharePoint files.
